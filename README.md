# NovelForge

**AI 无人值守长篇小说创作引擎** — 把写小说当作软件工程来做。

> 声明式配置 + 流水线调度 + 多维质控 + 断点恢复，让 Claude 从种子到成稿自动写完 300 章长篇。

## 当前状态

设计阶段。引擎实现尚未开始。

## 文档

- [设计文档](docs/plan/novelforge-design.md) — 架构、模块、流水线、实现路径

## 与 novels 的关系

本工程是**引擎**。使用本引擎驱动的小说项目（如 `novel`）以独立的 `novel-project.yaml` 声明形式存在，与本工程相互独立。

```
novelforge/   ← 本工程：引擎开发
```

## 实现计划

按设计文档第七节的三阶段推进：

1. **阶段一**：脚本编排器（Python/Node + Claude Code CLI），先跑通现有 300 章项目验证
2. **阶段二**：结构化审查、自动检查、质量趋势
3. **阶段三**：Web UI、多项目并行、EPUB/PDF 导出
